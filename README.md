# File System - Linux version

## Run project
```
git clone https://github.com/mohtzz/FileSystem-linux
```
```
npm install
```
```
tsc
```
```
npm run build
```
```
docker-compose up -d --build
```
```
go run main.go
```
### After this, app will be available on port 9015.
### You can also check Apache module working on port 8080
